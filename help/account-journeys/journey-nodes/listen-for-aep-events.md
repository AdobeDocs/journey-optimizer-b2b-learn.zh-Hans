# 监听AEP活动

在[!DNL Journey Optimizer B2B Edition]中侦听AEP事件允许您定义和使用帐户历程中的任何体验事件。 管理员可以配置基于Adobe Experience Platform (AEP)的事件定义，从而让营销人员能够创建对AEP Experience事件做出反应的帐户历程。 在帐户历程中使用AEP Experience事件包括两部分：

* 创建和发布AEP事件定义

* 在帐户历程中使用AEP事件
